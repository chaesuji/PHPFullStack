1. 개행
 - "\n" : 역슬래시 + n, 디버그 콘솔 창(소스코드)에서의 줄바꿈
 - "<br>" : 브라우저 상 줄바꿈
 - "<br>\n" : 소스코드 & 브라우저 돌시 줄바꿈

2. 출력 
 - echo / echo (출력할 내용) & echo 출력할 내용 & echo "출력할 내용"
 - print 

3. 변수 : 값을 담아서 코드 상에서 사용 / 정보를 저장하는 장소(공간)
 - 영어 대/소문자, 숫자, 특수문자(언더바(_)만 가능)로 이루어짐
 - $으로 시작하며 변수는 대/소문자 구분을 확실하게 하니 주의하기
 - 변수 이름을 설정할 때 언더바 외의 다른 특수문자, 숫자를 사용할 수 없음

$value - 변수로 사용 가능  / $123value, $##value - 변수로 사용 불가능

* 변수의 네이밍 기법 
1. 카멜 기법 : 낙타의 등을 연상하여 단어의 첫 글자를 대문자, 나머지는 소문자로 작성
예 > $TestName
2. 스네이크 기법 : 뱀을 연상하여 소문자로 작성하며 단어와 단어 사이를 언더바(_)로 이어 작성
예 > $test_name
